# wordle_solver_manuella-

## Description
Ce projet est un solver du jeu wordle realise en c.
Il permet de filtrer une liste de mots selon differentes conditions (lettre,
exclusion, substring)


## structure du projet

- bin : contient l'executable
- src : fichiers sources (.c )
- include : fichier header (.h)
- ressource : base de donnees des mots 
 

## compilation 

Pour compiler le projet, il faudra taper la commande suivante :

< ./bin/wordle_solver >
